About System

TactiSkin is a flexible textile-based smart sensing system designed for real-time pressure mapping.

  • 8×8 pressure sensing grid
  • Real-time heatmap visualization
  • Arduino-based acquisition
  • Designed for crash and helmet testing

Problem Statement

  • Lack of high-resolution pressure mapping
  • No localized impact detection
  • No real-time visualization
  • Limited sensor coverage

Proposed Solution

Flexible smart skin using Ecoflex, TPU, Velostat, conductive grid, and Arduino-based acquisition system.

System Flow

Impact → Pressure → Resistance → Voltage → Arduino → Data → Heatmap

Heatmap Visualization

Blue = Low, Green/Yellow = Medium, Red = High pressure zones.
