> SOLUTION_FLOW

[ Impact Input ]
External force applied
↓
[ Ecoflex + TPU Layer ]
Distributes pressure
↓
[ Velostat Layer ]
Resistance changes under pressure
↓
[ Thread Grid (8×8) ]
Captures sensing points
↓
[ Voltage Divider ]
Converts resistance to voltage
↓
[ Arduino Mega ]
Reads signals and scans grid
↓
[ Serial Communication ]
Transmits data (115200 baud)
↓
[ Data Processing ]
Maps pressure values
↓
[ Heatmap Output ]
Displays real-time heatmap

> WORKING_PRINCIPLE

> Impact applied → Ecoflex layer deforms and distributes force across grid.
> Velostat compresses → resistance decreases (~30kΩ to ~5kΩ).
> Each node acts as variable resistor → converted to voltage via 10kΩ divider.
> Arduino Mega scans rows sequentially → reads 64 sensing points.
> Full scan completed in ~40 ms (~25 Hz) → real-time pressure mapping.
> Filtered data sent via serial (115200 baud) → processed in Python.
> Heatmap output:
[BLUE] = low pressure
[GREEN/YELLOW] = medium
[RED] = high pressure

> CORE_FEATURES

> Real-time pressure mapping
> Flexible textile-based design
> 64 sensing nodes
> Lightweight and durable
> Low-cost system
