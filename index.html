HEATMAP_VISUALIZATION

Real-time pressure distribution mapped using sensor grid data.

Color gradient representation:

BLUE → Low pressure GREEN → Medium YELLOW → Moderate RED → High pressure

Enables clear identification of impact zones and force concentration.

> WORKING_PRINCIPLE

> Impact applied → Ecoflex layer deforms and distributes force across grid.
> Velostat compresses → resistance decreases (~30kΩ to ~5kΩ).
> Each node acts as variable resistor → converted to voltage via 10kΩ divider.
> Arduino Mega scans rows sequentially → reads 64 sensing points.
> Full scan completed in ~40 ms (~25 Hz) → real-time pressure mapping.
> Filtered data sent via serial (115200 baud) → processed in Python.
BLUE = Low Pressure
GREEN = Medium Pressure
YELLOW = Moderate Pressure
RED = High Pressure

> CORE_FEATURES

> Real-time pressure mapping
> Flexible textile-based design
> 64 sensing nodes
> Lightweight and durable
> Low-cost system
