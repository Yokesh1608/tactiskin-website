> PROPOSED_SOLUTION

Flexible textile-based smart skin using Ecoflex, conductive thread grid, Velostat sensing layer, and Arduino-based acquisition system.

> TECHNOLOGY

Layered smart textile sensing system combining flexible materials, embedded electronics, and real-time data processing.

> Ecoflex 00-30 + TPU coating → flexible impact-distributing surface
> Velostat film → piezoresistive sensing (30kΩ to 5kΩ under pressure)
> 8×8 conductive thread grid → localized pressure detection
> 10kΩ voltage divider → resistance-to-voltage conversion
> 0.1µF capacitors → noise filtering
> Arduino Mega (ATmega2560) → row-wise scanning + ADC
> Serial communication (115200 baud) → data transfer
> Python (Matplotlib) → real-time heatmap visualization
> PET + Kapton layers → electrical insulation
> Neoprene base (3mm) → structural support

> WORKING_PRINCIPLE

> Impact applied → Ecoflex layer deforms and distributes force across grid.
> Velostat compresses → resistance decreases (~30kΩ to ~5kΩ).
> Each node acts as variable resistor → converted to voltage via 10kΩ divider.
> Arduino Mega scans rows sequentially → reads 64 sensing points.
> Full scan completed in ~40 ms (~25 Hz) → real-time pressure mapping.
> Filtered data sent via serial (115200 baud) → processed in Python.

> Heatmap output:
[BLUE] = low pressure
[GREEN/YELLOW] = medium
[RED] = high pressure
